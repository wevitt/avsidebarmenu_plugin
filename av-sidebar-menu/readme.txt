== Description ==

This plugin allows you to view , in a sidebar , a custom menu for each post or page

== Installation ==

1. Upload `av-sidebar-menu` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
