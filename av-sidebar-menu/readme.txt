and display it as a widget into a sidebar .

== Installation ==
* Carica la cartella `av-sidebar-menu` nella cartella  `/wp-content/plugins/`.
* Attiva il plugin attraverso il menu 'Plugins' in WordPress.
* Aggiungi il widget 'AV Sidebar Menu' nella sidebar e inserisci un titolo.
* Dalla pagina crea/edit seleziona il menu personalizzato da visualizzare nella sidebar.

* Upload `av-sidebar-menu` folder to the `/wp-content/plugins/` directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Inserte the widget AV Sidebar Menu into sidebar and digit a Title.
* Select from create/edit page the custom menu to show into sidebar.
